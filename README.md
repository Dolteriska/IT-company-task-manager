Important:
Users with elevated permissions (e.g., super users or users with extended roles) see the interface differently compared to regular users. Their role grants them additional capabilities across the platform, such as managing tasks, positions, and user accounts.

All full-page screenshots in this repository were taken from an administrator’s perspective. As a result, certain UI elements—such as Create and Edit buttons—may not be visible to regular users for security reasons.

Permissions overview:

Administrators (Coordinators, Managers for example) / Super Users can:
Create, edit, and delete tasks
Manage user roles and positions
Create and edit user accounts
Regular Users can:
View tasks
Update the status of tasks assigned to them
Create and manage task types
Regular Users cannot:
Create, edit, or delete other users
Create or delete tasks
Manage positions (create, edit, or delete)
